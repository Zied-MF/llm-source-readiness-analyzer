# Product Spec

## Product name
LLM Source Readiness Analyzer

## Goal
Analyze a webpage URL and score whether the page’s content structure is suitable for being used as a source by LLMs in generated answers.

## Target user
- SEO consultants
- Content teams
- Agencies
- Website owners

## MVP input
- One URL

## MVP output
- Overall score out of 100
- 10 subscores
- Issues detected
- Recommendations
- Chunks preview

## Core promise
Help users understand whether a page is structurally ready to be extracted, understood, chunked, and reused by LLMs as a source.

## Out of scope for MVP
- Authentication
- Payments
- Database
- History
- Competitor comparison
- Per-LLM scoring
